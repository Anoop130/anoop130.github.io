KaamChalau - AI Resume Optimizer

Status: Active Development Updated: 1 hour ago Type: Full-Stack AI Application

An AI-powered resume optimization tool that tailors your resume for specific job descriptions using multiple AI models (Groq Llama 3.3 70B, Claude). Features real-time PDF preview with zoom controls and professional LaTeX generation.

Overview

KaamChalau is a modern web application that helps job seekers optimize their resumes for specific job postings using state-of-the-art large language models. The tool analyzes job descriptions and intelligently tailors resume content to match required skills and qualifications.

Key Features

  • Multiple AI Models: Choose between Groq (Llama 3.3 70B) or Claude for resume optimization
  • Real-time PDF Preview: See your resume updates instantly with zoom controls
  • LaTeX Generation: Professional LaTeX output for high-quality formatting
  • Keyword Matching: Automatically identifies and incorporates relevant keywords from job descriptions
  • Custom Templates: Upload your own LaTeX templates for personalized styling
  • Modern UI: React frontend with responsive design and smooth interactions

Technology Stack

Flask Python React JavaScript Groq API Claude API LaTeX

How It Works

The application works in three main steps:

  1. Input: Upload your master resume and paste the target job description
  2. Processing: The AI analyzes both documents and identifies key skills, requirements, and qualifications
  3. Optimization: Generate an optimized resume with relevant keywords and tailored content
  4. Export: Download as PDF or get LaTeX source for further customization

Project Architecture

Backend: Flask-based REST API handling file uploads, AI model communication, and LaTeX compilation

Frontend: React application with Vite for fast development, featuring real-time PDF rendering

AI Integration: Modular design supporting multiple LLM providers (Groq, Anthropic)

Recent Updates
